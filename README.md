# LeanBloomberg
Plugin library for implementations of the LEAN classes with Bloomberg.

##### Installation:

 - Install into the same root folder as /Lean, this way the dependencies will link with relative pathing.
  ```
  /Lean 
  /LeanBloomberg
  ```
